Shift+Insert                # 插入系统剪贴板的内容，在 paste 模式下不生效
Ctrl+P                      # 自动补全之前出现过的字符串
Ctrl+N                      # 自动补全只有出现过的字符串
Ctrl+X Ctrl+L               # 自动补全一行
Ctrl+X Ctrl+F               # 在文件系统中搜索，自动补全文件名

Ctrl+R                      # 插入上次 复制 或删除的内容
Ctrl+R+*                    # 插入 * 寄存器中的内容

<C-v>u00a9                  # 输入 unicode 码为 00a9 的字符
